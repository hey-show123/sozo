# 技術スキル・実績アピール文書

## プロジェクト概要
AI英会話学習アプリ「SOZO」の開発における技術的実装とアーキテクチャ設計を担当。モバイルアプリ、管理画面、バックエンドシステムの全領域において、モダンな技術スタックを活用した統合的なソリューションを実現。美容業界に特化したカリキュラムで、実践的な英会話スキルの習得を支援。

## 技術スタック・専門領域

### 1. モバイルアプリ開発 (Flutter)
**使用技術:**
- **Flutter 3.x** / Dart による iOS・Android・Webのクロスプラットフォーム開発
- **状態管理:** Riverpod 2.6+ による効率的な状態管理とDI実装
- **アーキテクチャ:** Freezed + json_serializable によるイミュータブルなデータモデル設計
- **ルーティング:** GoRouter による宣言的なナビゲーション実装

**実装機能:**
- リアルタイム音声録音・再生機能 (Record + Just Audio)
- Azure Cognitive Services連携による高精度な発音評価システム
- OpenAI GPT-4o / TTS API統合によるAI会話機能
- マイクロアニメーション (Flutter Animate) による洗練されたUX
- オフライン対応 (SQLite + Shared Preferences)

### 2. Web管理画面開発 (Next.js)
**使用技術:**
- **Next.js 15.3** / React 19 / TypeScript による最新のSSR/SSGアプリケーション
- **UIコンポーネント:** Radix UI + Tailwind CSS による高品質なUI実装
- **データフェッチング:** TanStack Query による効率的なサーバー状態管理
- **認証:** Supabase Auth + JWT によるセキュアな認証システム

**実装機能:**
- レッスンコンテンツ管理システム (CRUD)
- ユーザー学習進捗ダッシュボード
- AI会話プロンプト設定・管理
- リアルタイム学習データ分析

### 3. バックエンド・インフラ (Supabase / PostgreSQL)
**使用技術:**
- **Supabase** (PostgreSQL + Realtime + Edge Functions)
- **データベース設計:** 30以上のテーブルによる正規化されたスキーマ設計
- **認証・認可:** RLS (Row Level Security) による細粒度アクセス制御
- **Edge Functions:** Deno ランタイムによるサーバーレス関数

**実装内容:**
- マルチテナント対応のデータベース設計
- 学習進捗追跡・分析システム
- ゲーミフィケーション機能 (XP、バッジ、リーダーボード)
- リアルタイム通知システム

### 4. AI・機械学習統合
**統合サービス:**
- **OpenAI GPT-4o:** 文脈を考慮した動的会話生成
- **OpenAI TTS:** 自然な音声合成 (6種類の音声モデル)
- **Azure Speech Services:** 音素レベルの発音評価・音声認識
- **Azure Cognitive Services:** プロソディ分析による流暢性評価

**実装特徴:**
- プロンプトエンジニアリングによる効果的な会話誘導
- ユーザーレベル適応型の難易度調整
- リアルタイムフィードバック生成
- 学習パーソナライゼーション

### 5. DevOps・品質管理
**実装プラクティス:**
- Git によるバージョン管理とブランチ戦略
- CI/CD パイプライン構築 (GitHub Actions)
- 自動テスト (Unit / Integration / E2E)
- パフォーマンス最適化 (< 2秒起動、< 100ms UI応答)

## 実装の詳細と技術的成果

### 高度なAI統合実装
- **OpenAI GPT-4o / GPT-4o-mini 統合**
  - 動的プロンプト生成システムによる文脈を考慮した会話
  - 感情パラメータ対応のTTS実装（6種類の音声、15種類以上の感情表現）
  - Whisper APIによる高精度音声認識
  - ユーザーレベル適応型の難易度自動調整

- **Azure Speech Services 統合**
  - 音素レベルの発音評価（Granularity: Phoneme）
  - プロソディ分析による流暢性評価
  - リアルタイム音声認識とフィードバック
  - 単語別・音素別の詳細な発音スコアリング

### 進捗管理・学習システムの実装
- **6段階の学習モード管理**
  - Vocabulary（単語学習）
  - Key Phrase（キーフレーズ練習）
  - Listening（リスニング）
  - Dialog（対話練習）
  - Application（応用練習）
  - AI Conversation（AI会話実践）
  
- **詳細な進捗トラッキング**
  - モード別の完了状態管理
  - 学習時間の累積記録
  - マスタリーレベル（0-5段階）
  - セッション別・レッスン別の詳細データ保存

### セキュリティ・認証実装
- **多層セキュリティアーキテクチャ**
  - Supabase Auth + JWTによる認証
  - RLS（Row Level Security）による細粒度アクセス制御
  - PKCEフローによるモバイル向け認証最適化
  - 30日間のセッション永続化と自動更新

- **データ保護**
  - APIキーの環境変数分離
  - リフレッシュトークンによる安全なセッション管理
  - ユーザーデータの暗号化保存

### データベース設計の詳細

  - JSONB型による柔軟なメタデータ管理
